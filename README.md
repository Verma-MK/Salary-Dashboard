# Salary Dashboard

## 🚀 Overview
The **Excel Salary Dashboard** is an interactive data visualization project designed to analyze and explore salary trends in data-related jobs. This dashboard helps job seekers, students, and professionals understand how factors like **job role, location, and employment type** impact salaries.

Built entirely in **Microsoft Excel**, this project demonstrates strong skills in **data analysis, visualization, and dashboard design**.

---

## 🎯 Objectives
- Analyze salary trends across different data job roles  
- Compare salaries by country and job type  
- Provide insights for better career decision-making  
- Showcase Excel-based data analysis skills  

---

## 🛠️ Tools & Skills Used

### 📉 Data Visualization
- Bar Charts (Salary comparison by job role)
- Map Charts (Geographical salary distribution)
- Clean and user-friendly dashboard layout

### ❎ Data Validation
- Dropdown filters for:
  - Job Title  
  - Country  
  - Schedule Type  
- Ensures accurate and consistent user inputs  

---

## 📂 Dataset Details
The dataset contains real-world data science job information from 2023, including:

- 👨‍💼 Job Titles  
- 💰 Salaries  
- 📍 Locations  
- 🛠️ Skills Required  

---

## 📊 Dashboard Features

### 📊 Salary Analysis by Job Role
- Horizontal bar chart for easy comparison  
- Sorted by median salary (highest to lowest)  
- Highlights high-paying roles like Data Engineers and Senior positions  

### 🗺️ Salary Distribution by Country
- Interactive map chart  
- Color-coded salary ranges  
- Identifies global salary disparities  

### 🔍 Dynamic Filtering
- Users can filter data by:
  - Job Title  
  - Country  
  - Employment Type  
- Provides customized insights instantly  

---

## 💡 Key Insights
- Senior-level roles earn significantly higher salaries  
- Data Engineers and Scientists are among the top-paying roles  
- Salaries vary greatly by geographic location  
- Employment type impacts overall compensation
